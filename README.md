# Brilliant Angels Junior School Website

A modern, responsive website for Brilliant Angels Junior School in Kyengera, Uganda.

## Features

- Responsive design
- Interactive navigation
- Program information
- Admission details
- Contact information
- Online enrollment form

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts (Quicksand & Fredoka)
